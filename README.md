# Park Eco Trends Analysis

Archived repository for an environmental data analysis project originally completed during coursework on Codecademy.

The original notebook and datasets were not preserved. This repository remains as part of an ongoing engineering and learning archive documenting past analytical projects and future reconstruction efforts.


## Project Overview

This project investigates ecological and environmental trends related to park systems through exploratory data analysis and statistical visualization.

The analysis focuses on identifying patterns, environmental indicators, and relationships within ecological datasets.

## Project Goals

- Analyze environmental and ecological datasets
- Explore trends within park-related systems
- Visualize environmental patterns and distributions
- Practice data cleaning and exploratory data analysis workflows
- Develop analytical insights using statistical methods

## Technologies Used

- Python
- pandas
- NumPy
- matplotlib
- seaborn
- Jupyter Notebook

## Analytical Focus

The project includes:
- Data preprocessing
- Exploratory data analysis
- Trend analysis
- Statistical visualization
- Environmental pattern interpretation
